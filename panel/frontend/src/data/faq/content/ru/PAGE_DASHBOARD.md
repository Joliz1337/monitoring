# Дашборд

Главная страница — обзор всех серверов мониторинга в одной сетке. Быстро видно, что в порядке, что упало, где нагрузка на пределе.

## Что можно делать

- Смотреть статус каждого сервера (онлайн/оффлайн), текущий CPU, RAM, диск, скорость сети
- Организовывать серверы в **папки** (например, `Production`, `Staging`, `Dev`)
- Переключаться между режимами отображения: сетка, список, компактный/детальный вид
- Перетаскивать серверы мышью между папками (drag-and-drop)
- Быстро переходить на детальную страницу любого сервера кликом по карточке

## Ключевые понятия

**Статус-индикатор** — цветной кружок у имени сервера. Зелёный = агент отвечает и шлёт метрики, красный = недоступен (сеть, агент упал, firewall).

**Load Average** — среднее число процессов в очереди на CPU за последнюю минуту. Если значение выше числа ядер — сервер перегружен и запросы ждут.

**Traffic 30д** — суммарный трафик RX+TX за последние 30 дней. Нужен для понимания, сколько сервер реально прокачивает.

**Detail level** (standard / compact / minimal) — сколько данных показывать на карточке. Standard — всё, minimal — только имя и статус, помещается больше серверов на экран.

**Folder collapse** — сворачивание папки. Состояние запоминается локально в браузере, у каждого администратора своё.

## Частые вопросы

### Почему у сервера красный кружок, хотя сервер работает?
Сервер может работать, а агент мониторинга — нет. Проверьте: контейнеры запущены (`docker compose ps` в `/opt/monitoring-node`), порт 9100 открыт в firewall для IP панели, mTLS-сертификаты на месте (`/opt/monitoring-node/nginx/ssl/{ca,cert,key}.pem`). Если после переустановки ноды с новым NODE_SECRET панель не видит — нажмите **Regenerate Secret** в карточке, чтобы выдать свежий сертификат с актуальной CA-подписью.

### Как переместить сервер в папку?
Перетащите карточку сервера мышью в заголовок нужной папки. Если папок ещё нет — сначала создайте кнопкой «Создать папку» в правой части заголовка.

### Что делать, если Load Average красный?
Открыть сервер, посмотреть таблицу процессов на странице деталей, отсортировать по CPU — и выявить процесс, который забивает ядра. Возможно, нужно оптимизировать приложение или добавить ресурсов.

### Можно ли скрыть неактивные серверы?
Да. Отключённые серверы (кнопка power на странице Servers) не показывают метрики и остаются в списке серым цветом. Полностью скрыть их нельзя — для этого их нужно удалить.

### Что такое «компактный режим» и зачем он?
Уменьшенные карточки без графиков. Полезно, когда серверов 50+ и надо видеть всю инфраструктуру на одном экране.

## Если что-то не работает

- Дашборд пустой → проверьте, что серверы добавлены на странице [Серверы](#), мониторинг включён
- Данные не обновляются → посмотрите `Refresh interval` в настройках, по умолчанию 5 секунд
- Папки сбрасываются → это нормально при чистке кэша браузера, состояние папок хранится локально
