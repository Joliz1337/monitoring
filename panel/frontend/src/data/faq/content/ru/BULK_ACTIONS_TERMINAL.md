# Массовый терминал

Выполнение shell-команды на множестве серверов одновременно. Быстрее, чем заходить по SSH в каждый.

## Что делает
- **Exec mode** — одна команда напрямую в shell.
- **Script mode** — многострочный блок во временном файле (переменные, циклы, функции).
- Выбор shell: `sh` (совместимость) или `bash` (расширения).
- Timeout от 30 сек до 10 мин: по истечении SIGTERM, затем SIGKILL.
- Результат по каждому серверу: exit code (`0` — успех), stdout, stderr.

## Полезно знать
- `sudo` не нужен — агент работает от root, может даже мешать.
- Интерактивные команды не работают (`htop`, `vim`, `nano`, `passwd`).
- Цепочка команд: `&&` (если предыдущая успешна) или `;` (независимо).
- Опасно: команда повторится на всех сразу — внимательно проверяйте `rm`, `iptables`, `systemctl stop`, `reboot`.
- Падает по timeout на части серверов — увеличьте timeout, там может быть медленный диск/сеть.
- Часть серверов «pending» — агент не ответил, повторите операцию.
