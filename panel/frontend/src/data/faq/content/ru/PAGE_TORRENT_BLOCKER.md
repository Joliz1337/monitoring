# Блокировщик торрентов

Автоблокировка IP с торрент-активностью. Remnawave детектирует P2P, а панель банит эти IP через ipset на всех нодах. Remnawave только видит, блокирует — эта страница.

## Параметры
- **Poll Interval** — как часто воркер опрашивает Remnawave. По умолчанию 5 минут, минимум 1.
- **Ban Duration** — сколько IP держится в бане, потом освобождается. По умолчанию 30 минут.
- **Excluded servers** — серверы, на которые бан не применяется (dev-ноды, легальные раздачи).
- **Вебхук-предупреждение** — перед баном POST (только HTTPS) на ваш сервис с данными пользователя и IP, затем грейс-период, и только потом бан.
- Кнопка **Poll Now** — принудительный цикл сразу.

## Статус последнего цикла
- `success` — IP забанены.
- `no_reports` — торрент-активности не было (всё спокойно).
- `error` — сбой, обычно Remnawave недоступен или неверный токен.

## Полезно знать
- Если задан секрет, тело вебхука подписывается заголовком `X-Signature: sha256=HMAC`.
- Если вебхук не доставлен или нет telegram_id — IP всё равно банится после задержки.
- Не конфликтует с Blocklist: отдельные списки ipset с разными TTL.
- IP банятся, но подключаются — проверьте, что нода применила ipset (`ipset list`).
