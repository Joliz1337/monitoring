# Remnawave

Интеграция с внешним Remnawave: панель собирает онлайн-статистику VPN-пользователей и ищет HWID-аномалии (одно устройство под разными аккаунтами).

## Что показывает
- Онлайн-статистику: уникальные пользователи, IP, устройства, ноды
- Список пользователей и их активность
- HWID-аномалии — устройства, с которых заходят под разными аккаунтами
- Кнопка **Collect Now** — принудительный сбор данных, не дожидаясь интервала

## Параметры
- **API-токен** Remnawave с правами чтения — без него страница пустая.
- **Collect Interval** — как часто collector опрашивает Remnawave. По умолчанию ~60 секунд.
- **Exclusion list** — UUID/HWID, которые collector пропускает (тестовые пользователи).

## Полезно знать
- Вся статистика считается только по тем, кто **сейчас онлайн**, а не за период.
- HWID-аномалиям нужно 10+ минут сбора, иначе список пуст.
- Если статистика пустая — проверьте токен, статус collector (`running`) и доступность URL Remnawave из панели.
