# Внешние источники блок-листа

Готовые списки вредных IP, которые панель раз в час скачивает по URL и раскатывает на ноды через ipset.

## Популярные источники
- **Spamhaus DROP** — заведомо вредные сети, ботнеты и спам: `https://www.spamhaus.org/drop/drop.txt`
- **FireHOL Level 1** — агрегат из множества списков: `https://iplists.firehol.org/files/firehol_level1.netset`
- **Team Cymru BOGONS** — сети, которых не должно быть в интернете: `https://www.team-cymru.org/Services/Bogons/fullbogons-ipv4.txt`

## Формат файла
Обычный текст, по одному IP или CIDR на строку. Строки с `#` и пустые игнорируются.

```
1.2.3.0/24
5.6.7.8
10.20.30.0/23
```

## Полезно знать
- **direction** задаёт, что блокировать: входящий (`in`) или исходящий (`out`) трафик с этих IP.
- Если источник стал недоступен, его статус — `error`, но старые записи остаются в ipset, и блокировка продолжает работать.
- Объёма не бойтесь: ipset спокойно держит миллионы записей.
