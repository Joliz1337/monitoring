# Учёт ресурсов

Отслеживает потреблённые часы, квоты и даты истечения по каждому серверу. Удобно для подсчёта затрат и бюджета, особенно с Yandex Cloud.

## Что показывает
- **Quota** — лимит часов за период (обычно месяц) и сколько уже потрачено.
- **Hours used** — потрачено в текущем периоде, от даты начала до сейчас.
- **Expiry date** — когда квота сбрасывается. Это не удаление сервера.
- **days left** — дней до expiry: красный если просрочено, жёлтый < 14, зелёный > 14.

## Extend и TopUp
- **Extend** — сдвигает дату expiry вперёд, часы не трогает.
- **TopUp** — добавляет часы к квоте, дату не трогает.
- Нужно и срок, и часы — делайте оба действия по очереди.

## Полезно знать
- Исчерпанная квота — только предупреждение (показывает 100%), сервер продолжает работать. Жёсткая остановка — отдельно через алерт.
- **YC Account** автоматически тянет часы из Yandex Cloud; без него ввод ручной. Серверы вне YC заводятся только вручную.
- Не синхронизирует — проверьте OAuth токен (YC их перевыпускает), Folder ID, права `viewer` и доступ к `*.api.cloud.yandex.net`.
