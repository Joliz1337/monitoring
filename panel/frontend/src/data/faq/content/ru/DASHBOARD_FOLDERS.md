# Папки на дашборде

Группировка серверов в логические папки, чтобы быстро находить нужные ноды среди десятков и сотен.

## Что делает
- Создать папку, переименовать, удалить (серверы при этом не удаляются).
- Перетащить сервер мышью в папку, поменять порядок папок.
- Свернуть/развернуть папку, чтобы экономить место на экране.

## Полезно знать
- Состояние папок (список, порядок, свёрнутость) хранится **локально в браузере** (localStorage): у каждого администратора своя структура, между устройствами не синхронизируется, чистка кэша сбрасывает всё.
- Один сервер может быть только в одной папке.
- Подпапок нет — только один уровень. Для иерархии используйте имена вроде `EU-Production-VPN`.
- Массового переноса нет — серверы перетаскиваются по одному.
- Сервер пропал из папки после перезагрузки — сбросился localStorage или браузер в приватном режиме.
