# Telegram-уведомления

Отправка алертов в Telegram — основной канал оповещений. Быстро настраивается, работает на любом устройстве.

## Как настроить

1. Открыть Telegram, найти **@BotFather**
2. Написать `/newbot`, придумать имя (`MyMonitoringBot`)
3. Получить **Bot Token** (вид: `1234567890:AaBbCc...`)
4. Вставить токен в поле **Bot Token** на этой странице
5. Создать группу в Telegram (или использовать личный чат), добавить туда бота
6. Получить **Chat ID**: написать **@userinfobot** (для личного) или переслать сообщение из группы в **@RawDataBot**
7. Вставить Chat ID в соответствующее поле
8. Нажать **Test Send** — должно прийти тестовое сообщение

## Ключевые понятия

**Bot Token** — уникальный секретный ключ вашего бота. Никому не показывать: с ним можно отправлять сообщения от имени бота.

**Chat ID** — идентификатор чата (личного или группового), куда бот будет писать. Для групп — отрицательное число (`-1001234...`), для личных — положительное.

**Test Send** — отправляет одно сообщение «ping» через текущие настройки. Если оно пришло — всё работает.

## Частые вопросы

### Бот добавлен в группу, но сообщения не приходят
В группах бот по умолчанию в «privacy mode» — видит только команды. Для отправки это не важно, но если через `getUpdates` Chat ID не находится — отключите privacy через `/setprivacy` в @BotFather.

### Как получить Chat ID проще всего?
1. Добавьте бота в группу
2. Напишите в группе любое сообщение
3. Откройте `https://api.telegram.org/bot<TOKEN>/getUpdates`
4. Найдите `"chat":{"id": ...}` — это и есть Chat ID

### Можно ли отправлять в несколько чатов?
В текущей реализации — один чат на всю панель. Если нужно несколько — используйте группу с нужными участниками.

### Почему уведомление пришло один раз, а потом тишина?
Скорее всего, `cooldown` триггера не даёт слать повторно. Или условие перестало соблюдаться (сервер вернулся в норму).

## Если что-то не работает

- Test Send → ошибка 401 → неверный токен
- Test Send → ошибка «chat not found» → неверный Chat ID или бот не в чате
- Test Send → timeout → панель не имеет интернета или блокировка к api.telegram.org
