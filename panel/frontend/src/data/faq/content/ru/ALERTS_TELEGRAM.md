# Telegram-уведомления

Основной канал оповещений. Быстро настраивается и работает на любом устройстве.

## Как настроить
1. В Telegram откройте **@BotFather**, команда `/newbot`, придумайте имя.
2. Получите **Bot Token** (вид `1234567890:AaBbCc...`) и вставьте в поле Bot Token.
3. Создайте группу или используйте личный чат, добавьте туда бота.
4. Узнайте **Chat ID**: через **@userinfobot** или открыв `https://api.telegram.org/bot<TOKEN>/getUpdates` и найдя `"chat":{"id": ...}`.
5. Вставьте Chat ID и нажмите **Test Send** — должно прийти сообщение.

## Полезно знать
- Chat ID группы — отрицательное число (`-1001234...`), личного чата — положительное.
- Bot Token секретный: с ним можно писать от имени бота, никому не показывайте.
- Чат один на всю панель. Нужно несколько получателей — используйте группу.
- Ошибки Test Send: `401` — неверный токен, `chat not found` — неверный Chat ID или бот не в чате, timeout — нет доступа к api.telegram.org.
