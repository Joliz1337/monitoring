# Встроенный терминал

Выполнение команд на сервере без захода по SSH. Удобно для быстрых проверок.

## Что делает
- **Exec mode** — одна команда в текущем shell.
- **Script mode** — многострочный скрипт во временном файле (переменные, циклы, функции).
- Выбор shell: `sh` (совместимый) или `bash` (расширения).
- Timeout от 30 сек до 10 мин: по истечении SIGTERM, затем SIGKILL.
- Полный вывод: stdout, stderr, exit code (`0` — успех, остальное — ошибка).

## Полезно знать
- `sudo` не нужен — агент работает от root.
- Интерактивные команды не работают (`htop`, `top`, `vim`, `nano`, `less`, `passwd`). Замены: `cat`/`head`/`tail` вместо `less`, `ps` вместо `top`, `sed`/`echo > file` вместо `vim`.
- Очень большой вывод (MB+) обрезается — добавляйте `| head -100`, `| tail -50`, `| wc -l`.
- Exit code `127` — команда не найдена, укажите полный путь или поставьте пакет.
- Отменить запущенную команду нельзя — перезагрузка страницы не остановит её на сервере.
- История команд хранится в браузере (localStorage), у каждого администратора своя.
